# Java Gradle Project

This is a sample Java application project built with Gradle.

## Prerequisites

- Java 21
- Gradle 8.8

## Build

To build the project, run the following command:

    ./gradlew build

## Run

To run the project, run the following command:

    ./gradlew run

## Test

To run the tests, use the following command:

    ./gradlew test
